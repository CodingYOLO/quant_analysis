## 0. 目标与定位

每天**盘后数据齐全后自动**运行多套**内置科学策略**，产出推荐股池，含：量化数据 + 分析理由 + 板块情况 + 多路置信度 + T+1/3/5 前向追踪。

**与「因子选股」的区别**：因子选股=用户临时手选因子 AND 即时筛；选股池=固定内置策略每日自动跑 + 多路交叉验证 + 前向追踪。

**铁律**：真实计算、可溯源、无 mock；缺数据显式标注；理由类文字走 LLM 接地（只解释因子、不编造、附来源），数字类全部规则计算。

## 1. 标的池与基础过滤（所有策略共用）

- 全市场 A 股，剔除：ST/*ST、退市风险、当日停牌、上市<60 交易日（次新不足均线）。
- **基础风控**：当日绝对涨跌幅 ≤ 9.3%（防情绪高潮追高，板块感知涨停除外）；成交额 ≥ 1 亿；市值 200–5000 亿（沿用 .env MIN/MAX_MARKET_CAP）。
- 价格/均线统一用**前复权**（与宽表广度一致，避免除权污染）。

## 2. 信号精确定义（消除"突破/回调信号"的模糊）

| 信号 | 精确定义（可算） | 复用函数 |
|---|---|---|
| **站上均线** | 收盘 ≥ MA20 且 收盘 ≥ MA60（前复权） | `above_ma` |
| **MA60 斜率>0.5%** | (MA60_今 − MA60_{5日前}) / MA60_{5日前} > 0.5%，回看 N=5【可配】 | `ma_slope` |
| **突破信号** | 收盘创近 20 日新高 **或** (MACD 金叉 且 量比≥1.2) | `macd_golden_cross`/自算新高/量比 |
| **回调信号(缩量回踩)** | 收盘在 MA20 ±3% 区间 且 缩量(量比<0.8) 且 `pullback_quality_score ≥ 50` | `pullback_quality_score`/`is_shrink_volume` |
| **反转形态** | 近期低位 + 当日阳线/下影线 + 近 3 日主力资金累计 ≥ 0 | `has_lower_shadow`/`calc_rps` |
| **人气靠前(代理)** | 全市场换手率排名 ≤ 1500（换手率代理人气，daily_basic，可达可回补） | `build_popularity_proxy` |
| **昨日涨停溢价率(前瞻预警)** | 昨日涨停股今日开盘均涨幅 < -1% → 全局风控降级（你文档新增项） | `limit_list_d`+daily open |

## 3. 五大内置策略（条件精确化，纳入你文档 6 项修订）

| # | 策略名(英文键) | 硬条件 | 加分/软条件 | 解读 |
|---|---|---|---|---|
| ① | **主升突破** `breakout` | 站上MA20/60 + MA60斜率>0.5% + 突破信号 + (涨幅<9.5% 或 换手>3%) + 人气≤1500 | 主力净流入、控盘度(加分不否决)、RPS50≥70 | 趋势确立加速拉升 |
| ② | **资金回调低吸** `pullback` | 站上MA20/60 + MA60斜率>0.5% + 回调信号 + 近3日主力资金净流入 + 人气≤1500 | 人气回落≤1.2、资金衰减≤0.6（未溃散） | 右侧缩量回踩错杀 |
| ③ | **资金跟随** `flow_follow` | 站上MA20/60 + MA60斜率>0.5% + 回调信号 + 换手>3% | **全市场成交额前6** | 核心龙头二次反扑 |
| ④ | **反转低吸** `reversal` | 站上MA20/60 + MA60斜率>0.5% + 7日涨幅≤5% + 近3日主力资金≥0 + 反转形态 | 控盘度(加分) | 底部反转初现 |
| ⑤ | **主题选股** `theme_pick` | 属于 Tab1 热门主题(heat≥阈值,趋势 up/趋势) 的强势龙头(RPS50高/主力流入/未过热) | 题材梯队发酵(涨停数3日递增/连板≥2) | 题材驱动龙头 |

> 修订全纳入：60日斜率(全策略)、换手过滤(①③)、控盘度降为加分(①④)、删主题历史否决、梯队发酵强度(⑤)、昨日涨停溢价率前瞻(§5风控)。

## 4. 多路命中与置信度（吴川核心，截图印证）

### 4.1 来源标签（一只票可多路命中）
- `规则候选`：通过基础+技术规则过滤（最宽）。
- `规则已选`：规则候选中再过强弱/资金硬门槛（RPS50≥70 等）。
- `策略候选`：命中 ①~④ 任一策略。
- `主题选股`：命中 ⑤。
- `最关注`(★)：多路命中 + 风控通过 + 给完整交易计划（最终推荐）。

### 4.2 置信度公式【需校准 P1】
截图实证：策略模型命中(策略精选/Entry)权重最高 > 主题选股 > 纯规则候选。
```
conf_base = 0.45·命中策略(①~④,任一即1，多个取强)
          + 0.20·主题命中(⑤)
          + 0.15·规则已选
          + 0.10·规则候选
conf = clip( conf_base + 0.25·factor_score_norm − 风险扣分, 0, 1 )
```
- factor_score_norm = 个股综合因子分(RPS/资金/形态/均线)归一到 0-1。
- 风险扣分：命中避雷(减持/立案/问询)、过热(7日>15%且换手高)、大盘退潮。
- 校准目标（对齐截图）：策略+Entry → ~0.70-0.85；主题+蓄势 → ~0.40-0.50；纯规则候选 → ~0.25-0.40。

### 4.3 阶段 phase（个股级）
`Entry`(突破/反转,可入场) / `蓄势回踩`(回调待确认) / `观察`(信号弱/过热)。由命中策略与信号派生。

## 5. 风控（盘后总监，沿用你文档）

- **大盘 gate**：复用 `a_market_gate` 4 态（强势/震荡/弱势/数据缺失）+ 风险预算系数。
- **昨日涨停溢价率<-1%** → 风险系数下调至 0.6、池子压缩、买入信号标"需复核"（你文档新增前瞻项）。
- **单主题数量上限**：同一主题入选 ≤ N 只（默认 3），防集中度过高。
- **个股避雷**：命中立案/减持/问询/退市/业绩预亏 → 降级"仅观察"或剔除（复用 `market_extras`/`news_guard` 博查舆情）。
- 仓位/买点/止损止盈：复用 `calc_buy_zones`/`calc_stop_loss_price`/`calc_take_profit_prices`/`calc_position_pct`。

## 6. 每日盘后自动流程

```
18:00  wide(宽表+人气)      ← 已有
18:30  run(完整选股流水线)   ← 已有(吴川三层)
18:40  stock-pool(新)        ← 跑①~⑤ → union去重 → 多路置信度 → 风控 → 落库 stock_pool 表
        ↳ forward 回填：每日开盘后/盘后用 forward_tracker 回填昨日及历史 T+1/3/5
```
- 前端只读 stock_pool 表 + performance（T+N）；不实时算。
- 理由(LLM 接地，复用 detail_common 博查+反编造)按需或盘后批量为"最关注"股生成。

## 8. Tab2 页面（对齐截图）

- **筛选条**：日期 / 代码·名称·主题 / 来源(多选) / 策略(多选) / 主题类型 / 排序(默认关注优先)；收益基准日可选。
- **列**：代码 / 名称 / 主题(带热度) / 来源(多标签) / 策略 / 阶段 / 仓位 / 置信度 / 买点价 / T+1 / T+3 / T+5 / 关注★。
- **行展开**：完整因子 + 分析理由(LLM) + 板块情况(宽表) + 交易计划 + 避雷提示。
- 顶部：总数 / 关注数 / 收益基准日。收盘后停实时、显"收盘态"。
