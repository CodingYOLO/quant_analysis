## 二、吴川方法逐条解析（供参考的"量化标准"）

### 2.1 市场环境分桶（高位分歧判定）
吴川用一组**字段化证据**定调市场，阈值清晰：
- `above_ma5_ratio_pct` 68.07% → 48.88%（**骤降**）= 普涨转普跌
- `down_count` 3534（指数涨但下跌家数多）= 指数与个股背离
- `limit_up_count` 137 → 95（涨停萎缩）
- `highest_board` 3（高度板低）= 情绪持续性弱
- 综合 → **阶段=高位分歧 / 分桶=震荡 / 置信度0.75 / signal_level=neutral**

> 借鉴点：**"指数涨 + 下跌家数多 + 站5日线骤降 + 涨停萎缩 + 高度板低" = 高位分歧**，这套组合判据值得固化为一个可复用的判定。

### 2.2 主控风控（decay_ratio 驱动 risk_off）
- `市场标签=衰退 / 风控级别=risk_off / 情绪分=54.80 / 衰减占比=0.71 / 单票仓位上限=0.10`
- **decay_ratio=0.71** 是核心：71% 板块退潮 → 直接 risk_off + 压缩仓位与关注数。

### 2.3 板块阶段识别（trend_score + 3日变化 Δ）
- 每个板块：`趋势评分(0-100) + 窗口变化(3日Δ)` → 阶段 `main_up/low_buy/decay/趋势`
- **Δ 是关键**：休闲食品 Δ=-44.25（急速退潮）、半导体 Δ=+51.80（主升加速）。Δ 区分"加速 vs 衰减"。
- 产出三榜：**趋势低吸候选 / 主升候选 / 退潮预警 TOP10**。

### 2.4 历史反馈作前向闸门
- `PCB板 history_t1_win_rate=0.20 样本=30 → 历史标签=避免追涨`（即便当天反弹）
- `光通信 样本=2 T+1胜率=100% → 样本不足`（不轻信小样本）
- **用历史胜率 + 样本量双重把关追涨。**

### 2.5 全市场信号回测（10万+样本）
| 信号 | T+1样本 | T+1胜率 | T+1均收益 |
|---|---:|---:|---:|
| 事件动量 | 10888 | **53.58%** | **+0.90%** |
| 区间均值回归 | 115066 | 46.65% | -0.08% |
| 低位吸筹 | 309037 | 46.18% | -0.02% |
| 回踩反转 | 44152 | 43.38% | -0.10% |
| 资金跟随 | 9385 | 44.17% | -0.17% |

> **冷静解读**：吴川多数信号 T+1 胜率 43-47%（接近随机），只有"事件动量">53%。说明他的护城河不是"必胜公式"，而是**全市场回测验证 + 严格风控纪律**。本系统方向正确。

### 2.6 个股级标准（高胜率个股报告）
- **来源归因链**：`pullback > market_pullback_flow > 全市场回调吸筹`，多来源重叠=高置信。
- **次日验证（量化可勾选）**：开盘价≥昨收×(1-2%)、分时量能不萎缩>30%、资金1日维持正、不低开低走。
- **多条件止损**：跌破MA20 **或** 放量下跌(量>1.5×vol_ma20且收阴) **或** 资金1日转负超-5；**题材级**：板块资金流出超3日/龙头破位。
- **人气排名动态**：人气最佳=84 / 当前=100 / 改善_3日=81，用"排名改善"作买点因子。
- **价格区间 + as_of + 推理依据**：保守=20日VWAP/5日线，激进=今日收盘，均带 as_of 时间戳。

## 三、提质方案（按优先级分三档）

### 🥇 第一档（高价值·低成本·先做）

#### A. `decay_ratio` 全市场板块衰减占比
- **现状**：有每板块 phase，无聚合。
- **做法**：`decay_ratio = 退潮板块数 / 总板块数`；接入大盘情绪「市场环境」与 `_classify_market_regime`，`>0.6` → 偏防守标签 + 提示。
- **落点**：`sector_analyzer` 聚合 + `market_sentiment` 展示 + e_report 市场环境引用。
- **为什么**：一个数量化"指数涨但个股普跌"的高位分歧，信号强、改动小。

#### B. 板块阶段 Δ + 主升/低吸/退潮 TOP 榜
- **现状**：sector_scope 三栏诊断，有 trend_score，无 3日Δ、无 TOP 榜。
- **做法**：补 `trend_score_delta_3d`；在板块全景页加三个 TOP 榜（主升=评分高且Δ>0；低吸=结构好且短期回踩；退潮=评分低且Δ大幅为负）。
- **落点**：`sector_scope.py`（已重构，扩展顺手）+ `sectorscope.html`。
- **为什么**：Δ 是"趋势的变化率"，区分加速/衰减，是吴川阶段识别的精髓。

#### C. 历史主题胜率 → 前向闸门
- **现状**：`get_theme_win_rates` 已算、e_report 已读，但未用于否决/降级。
- **做法**：主题历史 T+1 胜率 `<35% 且样本≥10` → 该主题个股自动打「⛔避免追涨」标签并降置信；样本<5 标「样本不足，谨慎」。
- **落点**：`stock_pool.py` / `c_stock_selection.py` 风控段 + 报告标注。
- **为什么**：已有数据，只差用起来；直接复刻吴川"PCB板20%胜率→避免追涨"。

### 🥈 第二档（高价值·较高成本·按路线图）

#### D. 全市场信号回测库（10万+样本）
- **现状**：仅炸板率→大盘代理。
- **做法**：按《优化路线图》建"前复权个股序列 + 事件驱动回测引擎"，对每个信号/策略做全历史全市场回测，产出 T+1/3/5 胜率+均收益+样本量；在「策略验证」页展示。
- **为什么**：这是真正拉开质量、形成护城河的部分。吴川报告已实锤此方向。

### 🥉 第三档（质量打磨·低成本·随手做）

- **E. 多条件 + 题材级止损**：止损升级为「MA20破位 / 放量下跌(量>1.5×MA20量且收阴) / 资金1日转负」+ 题材级（板块资金流出超3日）。落点 `factors.core` + 交易计划。
- **F. 次日验证条件量化**：报告提示词固化 4 条可勾选规则（开盘价、分时量能、资金1日、不低开低走）。落点 quick_report / e_report 提示词。
- **G. 个股人气排名改善_3日**：因子表/选股池补"人气排名3日改善值"。落点 `screener` / `popularity`。
- **H. 结构化证据字段**：关键结论附 `{field, value, description}`，机器可读可追溯。

## 四、执行计划（逐步）

> 原则：每步**小步快跑**——改 → 真实数据验证 → 部署 → 提交。遵循 DI/协议/可测试；缺数据显式标注，禁 mock。

| 步骤 | 内容 | 文件 | 验收 |
|---|---|---|---|
| **S1** | A. decay_ratio 聚合 + 接入市场环境/情绪页 | `sector_analyzer.py` `market_sentiment.py` `e_report.py` | 真实数据算出 decay_ratio 并展示；>0.6 触发防守标签 |
| **S2** | B. 板块 trend_score Δ + 主升/低吸/退潮 TOP榜 | `sector_scope.py` `sectorscope.html` | 板块全景页出现三 TOP 榜，Δ 正确 |
| **S3** | C. 历史胜率前向闸门（低胜率主题禁追涨） | `stock_pool.py` `c_stock_selection.py` | 低胜率主题个股自动标「避免追涨」 |
| **S4** | E/F. 多条件止损 + 次日验证量化 | `factors.core` 提示词 | 报告止损/次日条件更具体 |
| **S5** | G/H. 人气排名改善因子 + 结构化证据 | `screener` `popularity` 报告层 | 因子可筛、证据可溯 |
| **S6（大）** | D. 全市场信号回测库 | 见《优化路线图》回测引擎 | 策略验证页出全市场 T+1/3/5 胜率 |

每步独立提交，便于回溯与同步。第一档（S1-S3）预计 1-2 天可显著提质。

## 五、专家提醒

1. **不神化吴川**：其多数信号 T+1 胜率 45-47%（接近随机），真正价值在**风控纪律 + 全市场回测验证**，而非必胜公式。
2. **先用已有数据**：第一档 A/B/C 大多是"已有评分/已有胜率，只差临门一脚"，性价比最高。
3. **回测引擎是护城河**：第二档 D 投入大但回报最高，决定能否从"看起来专业"升级到"经得起验证"。
4. **纪律不变**：所有改动走 `CompositeProvider`，禁 mock，缺数据显式标注，关键逻辑配单测。
